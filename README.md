# Formula 1 Data Analysis

## Überblick

Dieses Projekt analysiert Formel-1-Daten, um Teamleistungen, Fahrerstatistiken und Zusammenhänge zwischen Rennplatzierungen und Punkten zu untersuchen.

Das Projekt wurde mit Python und Jupyter Notebook im Rahmen einer universitären Datenanalyse erstellt.

---

## Verwendete Technologien

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook

---

```txt
formula1-data-analysis/

```

---

## Enthaltene Analysen

- Teamleistungen seit 2010
- Vergleich von Fahrerleistungen
- Korrelation zwischen Platzierungen und Punkten
- Analyse von Ferrari-Motor-Teams
- Statistische Auswertungen und Visualisierungen

---

## Wichtigste Erkenntnisse

- Mercedes und Red Bull dominierten die moderne Formel-1-Ära.
- Zwischen Platzierung und Punktzahl besteht eine starke Korrelation.
- Teams mit Ferrari-Motoren zeigen unterschiedliche Leistungsniveaus.
